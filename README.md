# project-rock-paper-scissors
The first project in The Odin Project curriculum, specificially in the foundations JavaScript. The project is the game Rock, Paper, Scissors.  

Simple logic work the 'magic'. The psuedocode for the playRound function was an excessive complication. All that is needed is that the code is evaluated such that if both human and computer choices are same, it is a draw. Or, if human's choice beat computer's human wins. Otherwise, computer bags all the wins.
